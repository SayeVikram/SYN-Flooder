# SYN-Flooder
My SYN Flooder for Linux/Unix systems


Setup:
   git commit https://github.com/SayeVikram/SYN-Flooder.git
   cd SYN-Flooder
   chmod +x syn.sh
   ./syn.sh
   
Use:
   Enter the IP address and the port you want to flood, along with the number of queries
